# MyStockScanner
Rule-based stock scanner with backtesting and paper trading for educational purposes. 
Stock Scanner is an educational platform designed to help users learn how market strategies work using scanners, backtests, and paper trading.

Users can:

Build their own strategies

Test them on historical data

Practice in live markets using virtual money

This platform does not provide trading tips or guaranteed results and is intended strictly for learning purposes.

This project is an educational stock analysis platform that includes:

Intraday Stock Scanner

Positional / Stock Scanner

Strategy-based Scoring Console

Historical Backtesting (simulation)

Paper Trading with virtual money

Portfolio Manager

All features are rule-based, user-controlled, and built only for learning and practice.
No investment advice, no buy/sell recommendations, and no real-money trading.
